Objeto Virtual de Aprendizaje - OVI
Programación Web 1


Con el fin de adquirir conocimiento y desarrollar destrezas en el uso del lenguaje Php y las Bases de datos, permitiéndonos avanzar en nuestro proceso de formación y el uso de las tecnologías.



Objetivos del curso


- Entender y comprender los conceptos básicos sobre la vinculación de una página web con una base de datos, en la operación con registros dentro de una tabla.

- Identificar conceptos como: HTML, PHP, SGBD, Base de datos, MySql para diferenciar sus usos y las aplicaciones que tienen.

- Aprender a través de un OVI, los temas sobre la vinculación de páginas web y bases de datos.
